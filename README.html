Feline Finances

Feline Finances is a virtual pet game for the Introduction to Programming prompt. The player creates a cat, keeps every care stat above zero, and manages the cost of pet ownership through shopping, chores, and vet care.

Judge Focus
  • Customization: name, cat type, and personality
  • Pet care: feed, play, rest, clean, and vet visits
  • Reactions: mood changes, health loss, and game-over on neglect
  • Cost of care: store purchases, spending totals, and chores for income
  • Documentation, save system, validation, and modular code structure

Rubric Match

Code Quality
  • Code is split into modular files inside source/scripts
  • Separate files handle UI, storage, models, assets, config, and game flow
  • The project uses classes, functions, dictionaries, lists, and conditionals
User Experience
  • Clear setup controls and simple mouse-based gameplay
  • Help, report, store, vet, chore, and delete-save overlays are built into the UI
  • Window resizing preserves the layout through scaled rendering
Validation And Storage
  • Name input is limited during setup
  • Save data is stored in JSON and validated before loading
  • Progress, inventory, spending, and achievements persist in the save file
Functionality
  • The user creates a custom pet before play begins
  • The pet can be fed, played with, rested, cleaned, and treated at the vet
  • Health is separate and only restored through paid vet visits
  • Money is spent on food, toys, grooming, and vet care
  • Money is earned through chores on the Taskboard
Presentable Report
  • The report overlay shows mood, survival time, money, spending, chores, and achievements
  • The main screen also shows live stats, inventory, total earned, and total spent
  • The README and source code are included for documentation

Main Features

Care Loop
  • Hunger, happiness, energy, cleanliness, and health bars
  • Mood changes based on current condition
  • If any stat reaches zero, the pet dies and the run ends
Money Loop
  • Whiskermart sells care items
  • Total earned and total spent are always visible
  • Expense categories track food, toys, grooming, and vet care
Extra Features
  • Care report with stats, spending, chores, and badges
  • Achievements and savings goal
  • Delete Save button with confirmation

How To Run

Run Steps
  1. Install Python 3.
  2. Install Pygame with pip install pygame.
  3. Open the project folder.
  4. Run python source/scripts/main.py.
Project Structure
  • Entry point: source/scripts/main.py
  • Core code: source/scripts/core
  • Gameplay loop: source/scripts/gameplay
  • UI and assets loader: source/scripts/interface and source/scripts/engine
  • Assets: source/assets
  • Save file: source/save_data.json

Technical Summary

Implementation
  • Python and Pygame
  • Classes, functions, lists, dictionaries, and conditionals
  • JSON save system with validation
  • Modular structure for game logic, UI, storage, models, and assets
